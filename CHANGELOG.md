# Changelog

Все заметные изменения скилла `yandex-wordstat`.
Формат — [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
версии — [SemVer](https://semver.org/lang/ru/) (`MAJOR.MINOR.PATCH`).

Правило: при изменении скилла — поднять версию здесь и в `version:` (frontmatter
SKILL.md), проставить git-тег `vX.Y.Z`. Журнал применений и правок — в `log/`.

## [Unreleased]

## [0.2.0] — 2026-06-10
### Added
- Методология анализа спроса (адаптирована из скилла `artwist-polyakov/polyakov-claude-skills`):
  - проверка интента через WebSearch (целевой / нецелевой запрос);
  - STOP-блок — уточнять регион и бизнес-цель перед анализом;
  - шпаргалка операторов Wordstat, точная частотность `"!слово !слово"`;
  - отчёт с разделением на целевые / нецелевые запросы с обоснованием.
- Тип `со_словами` (режим «Популярные») как 4-й тип данных; колонка `интент` в CSV.
- Механизм версионирования: поле `version:` в SKILL.md, этот CHANGELOG, git-теги `vX.Y.Z`.
- Журнал `log/` (применения и правки) + правила ведения в SKILL.md.
- Заметка о сессии: не закрывать браузер между прогонами; сохранять стейт в
  `~/.agent-browser/yandex-state.json` (логин переживает закрытия).
### Changed
- Роли уточнены: `со_словами` — основной пул семантики, `похожие` — шум для проверки.

## [0.1.0] — 2026-06-09
### Added
- Первая рабочая версия скилла: сбор частотностей и связанных запросов из
  Яндекс Wordstat через `agent-browser`.
- URL-навигация `?words=<фраза>&region=<geo-id>` (надёжнее ввода в поле и модала региона).
- Таблица проверенных geo-id: Беларусь 149, Минск 157, Москва 213, Россия 225.
- Кодирование кириллицы через `node encodeURIComponent`.
- Ручной логин в Яндекс, сессия `--session yandex` (стейт вне репо).
- Вывод: CSV (базовая/точная/похожая) + сводка в чат. Дизайн, план, smoke-тест.

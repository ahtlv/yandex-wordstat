# yandex-wordstat

Скилл для [Claude Code](https://claude.com/claude-code): собирает ключевые фразы,
частотности и похожие запросы из [Яндекс Wordstat](https://wordstat.yandex.ru)
для русскоязычного SEO. Работает через [`agent-browser`](https://github.com/vercel-labs/agent-browser).

## Что делает

По списку базовых фраз снимает:
- **базовую частотность** (широкое соответствие);
- **точную частотность** (оператор `"!фраза"`);
- блок **«Запросы, похожие на …»** — для расширения семантического ядра.

Результат — CSV + краткая сводка в чат. Каждая выгрузка складывается в
`research/<дата>-<тема>/`.

## Требования

- [Claude Code](https://claude.com/claude-code)
- [`agent-browser`](https://www.npmjs.com/package/agent-browser) в `PATH`
  (`npm i -g agent-browser`)
- Аккаунт Яндекса (логин — руками при первом запуске)

## Установка

Как субмодуль в каталог скиллов Claude Code, например:

```bash
git submodule add https://github.com/ahtlv/yandex-wordstat \
  ~/.claude/skills/yandex-wordstat
```

## Использование

```
/yandex-wordstat <фразы через ;> [--регион|--region <город>] [--тема <слаг>] [--out <csv>]
```

Пример:

```
/yandex-wordstat ателье штор; пошив штор на заказ --регион Минск
```

Регион по умолчанию — **Беларусь**.

## Безопасность

Скилл не хранит логины и пароли. Авторизация Яндекса проходится руками в окне
браузера при первом запуске; сессия и cookies сохраняются `agent-browser` в
`~/.agent-browser/` — **вне этого репозитория**. Поэтому репозиторий публичный:
клонируя его, вы получаете только инструкции, а работаете под своим аккаунтом.

Опционально можно включить шифрование стейта (`AGENT_BROWSER_ENCRYPTION_KEY`,
AES-256-GCM) и авто-протухание сессий (`AGENT_BROWSER_STATE_EXPIRE_DAYS`).

## Дисклеймер

Инструмент для личного подбора семантики. Соблюдайте условия использования
Яндекс Wordstat и разумные лимиты запросов — иначе капча/бан.
